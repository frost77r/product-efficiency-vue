# 产品效能分析 Vue 项目

校招 AI 实践作业前端实现，基于 Vue 3、TypeScript、Element Plus 和 ECharts 开发。项目使用作业资料中的产品、码表和订单样例数据生成本地数据源，可直接演示统计分析、订单清单钻取和运维智能助手页面。

## 功能模块

- 统计分析页：本地网、产品分类、订单状态、统计时间筛选。
- 环比指标卡：订单量、竣工量、在途单量、删单量、撤单量 5 个指标。
- 产品订单统计：按产品维度展示在途、竣工、异常、撤单、删单堆叠柱状图。
- 趋势图：订单总量、竣工量、在途单、删单量、撤单量 5 个折线图。
- 清单详情页：12 列订单明细、分页、条件回显、图表点击钻取。
- AI 综合分析：汇总当前页面数据生成 6 章节分析报告，支持复制、导出 HTML、浏览器打印 PDF。
- 运维智能助手：按“查数据 -> 查配置 -> 查代码”的顺序生成核查步骤，并支持 SQL 复制。

## 技术栈

- Vue 3
- TypeScript
- Vue Router
- Element Plus
- ECharts
- Vite

## 启动方式

```bash
npm install
npm run dev
```

浏览器访问终端输出的本地地址，默认脚本会绑定 `127.0.0.1`。

## 构建

```bash
npm run build
```

构建产物输出到 `dist/`。

## 数据说明

项目中的 `src/data/generatedData.ts` 由作业资料中的 SQL/CSV 示例数据生成，包含：

- 产品分类数据
- 本地网和订单状态码表
- 订单效能样例数据

当前项目是纯前端演示版本，不依赖后端服务。后续如果接入真实接口，可将 `src/utils/statistics.ts` 中的本地计算逻辑替换为接口请求。

## 页面路由

- `#/statistics`：产品效能统计分析
- `#/list`：订单清单详情
- `#/troubleshoot`：运维智能助手
